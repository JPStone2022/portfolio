Accessibility Statement

{# Use prose for nice typography #}

I am committed to ensuring digital accessibility for people with disabilities. I am continually improving the user experience for everyone and applying relevant accessibility standards.

Conformance Status

The Web Content Accessibility Guidelines (WCAG) defines requirements for designers and developers to improve accessibility for people with disabilities. This website aims to conform substantially with WCAG 2.1 level AA.

While I strive to adhere to the accepted guidelines and standards for accessibility and usability, it is not always possible to do so in all areas of the website, particularly with some embedded third-party content or complex interactive demos.

Technical Specifications

Accessibility of this website relies on the following technologies to work with the particular combination of web browser and any assistive technologies or plugins installed on your computer:

  • HTML
  • CSS (Tailwind CSS)
  • JavaScript

These technologies are relied upon for conformance with the accessibility standards used.

Accessibility Features Implemented

  • Semantic HTML: Using appropriate HTML5 tags (nav, main, section, article, etc.) to define page structure.
  • Keyboard Navigation: Efforts have been made to ensure interactive elements are navigable using a keyboard.
  • Focus Indicators: Visible focus rings are applied to interactive elements.
  • Alt Text: Alternative text descriptions are provided for meaningful images.
  • Responsive Design: The layout adapts to different screen sizes and orientations.
  • Color Contrast (Ongoing): We strive to maintain sufficient text and background color contrast, and background elements, including in dark mode.
  • Resizable Text: Text can be resized using standard browser controls without loss of content or functionality.

Feedback & Contact Information

I welcome your feedback on the accessibility of this portfolio website. If you encounter accessibility barriers or have suggestions for improvement, please let me know:

  • Contact Form: Use the Contact Form

I try to respond to feedback within a reasonable timeframe.
